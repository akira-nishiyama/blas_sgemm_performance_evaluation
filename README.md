# blass_sgemm_performance_evaluation
This repository evaluates float matrix production.

# Requiremets

cmake  
make  
gcc  
OpenMP  
OpenBLAS  
CUDA  

# Usage

Open jupyter notebook(plot.ipynb) and run all.  
The artifacts will be placed in build directory.
